Installation with Anaconda
--------------------------

.. code-block:: console

    $ conda create -n cima-goes-examples python=3.8
    $ conda activate cima-goes-examples
    $ pip install cima.goes

    (Run "conda install -c conda-forge cartopy" if necessary)
